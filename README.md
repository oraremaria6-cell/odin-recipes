# Odin Recipes

This project is part of the Odin Project's curriculum. It is a simple recipe website where I'll practice using HTML, CSS, and Git. Throughout this project, I will demonstrate the ability to:

- Set up a Git repository
- Clone and manage the repository
- Create and organize files locally
- Write a basic README file to document the project

Once finished, I'll have a functional recipe website that showcases the skills learned in the Odin Project.
